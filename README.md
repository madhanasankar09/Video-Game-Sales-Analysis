# Video Game Sales & Engagement Analysis

This project analyzes video game sales and engagement using Python, SQL, and Power BI.

## Files
- app.py – Python script for data preprocessing and database creation
- games_clean.csv – Cleaned games dataset
- vgsales_clean.csv – Cleaned sales dataset
- videogame.db – SQLite database
- Video Games.pbix – Power BI dashboard
- Video Game Sales and Engagement Analysis.pdf – Documentation and insights

## Workflow
1. Clean raw data using Python (Pandas).
2. Store cleaned data in SQLite database.
3. Connect Power BI to the database.
4. Build dashboards for:
   - Genres
   - Platforms
   - Sales trends
   - Ratings & Wishlist
5. Add insights in each Power BI page.

## Tools Used
- Python (Pandas)
- SQLite / SQL
- Power BI Desktop
